# 24点游戏求解器

## 简介
这是一个24点游戏的求解程序，可以判断输入的4个数字是否能通过四则运算（加、减、乘、除）和括号得到24。

## 功能特点
- ✅ 支持输入4个1-10的数字
- ✅ 自动尝试所有可能的运算组合
- ✅ 如果能得出24，给出一种具体的运算方式
- ✅ 如果不能得出24，会明确提示
- ✅ 支持连续查询，可以多次输入
- ✅ **单文件exe，无需安装Python或任何依赖**

## 使用方法

### ✨ 推荐方法：直接运行exe（最简单）
双击运行 `dist\game_24.exe`

**这是单文件版本，包含了所有依赖，可以在任何Windows电脑上运行！**

### 方法二：使用批处理文件
双击运行 `运行24点游戏.bat`

### 方法三：命令行运行
```bash
python game_24.py
```

## 操作说明
1. 程序运行后，输入4个数字（用空格分隔），例如：`3 4 5 6`
2. 程序会自动计算并显示结果
3. 如果要退出程序，输入 `q` 即可
4. 程序结束时会提示按回车键退出

## 示例

### 示例1：可以得出24
```
请输入4个数字（用空格分隔，输入q退出）：1 4 6 8

输入的数字：[1, 4, 6, 8]
------------------------------------------------------------
✓ 可以找到解法！
  表达式：((1-4)+6)*8 = 24
  验证：((1-4)+6)*8 = 24
```

### 示例2：无法得出24
```
请输入4个数字（用空格分隔，输入q退出）：1 1 1 1

输入的数字：[1, 1, 1, 1]
------------------------------------------------------------
✗ 无法得出24
  这4个数字的所有组合都无法通过四则运算得到24
```

- 打包工具：PyInstaller (--onefile模式)

- 支持的括号组合：5种基本模式
- 打包方式：单文件独立可执行文件（包含所有Python依赖）
